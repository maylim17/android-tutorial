Intermediate > Statements and Blocks


To declare a variable:
   dataType variableName;

To assign a value to a variable that's already declared:
   variableName = assignedValue;

To declare and assign variable tgt in one line:
   dataType variableName = assignedValue;


To declare and assign elements to an array:
   dataType[] arrayName = {element1, element2, ...};


To declare a method:
   public outputType methodName (inputType inputName) { ... }


To leave comment in any point of the code:
   // this is a comment statement
