Syntax > for (a; b; c) { }
   a- initialize counter
   b- condition check at each loop start
   c- post-operation at each loop end

1|   for (i = 0; i < 5; i++) {
2|      machine.dispenseCoin(0.5);
3|   }




Syntax > while (a) { }
   a- condition check at each loop start

1|   int counter = 1;
2|   
3|   while (counter <= 8) {
4|      machine.dispenseCoin(0.1);
5|      counter = counter + 1;
6|   }
