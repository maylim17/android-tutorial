Fun Facts App > Create XML Layout


1. Switch to Design tab at the bottom of the code editor window

This allows us to use the graphical editor to create our activity layout (XML file)



2. Edit the "Hello world!" textView

Click the textView on the phone display and replace the text as "@string/did_you_know". Also set its id = "header". You will get an error but we will correct it next



3. Resolve the error with strings.xml

Android's programming convention is to define all text values in a file called strings.xml. Navigate to the file in the project directory, and add the line as shown


This lets us set all text values in one place, and the same text values can be referenced from multiple places.



Switching to the Text tab, we see the XML code of our change made earlier (line 12).


4. Add a second textView to display our fun fact

Switch to the Design tab, select a Plain TextView under Widgets, drag and position it on the phone display


In the Properties window on the right, set id = "funFact", text = a fun fact about yourself, textSize = "24sp". Select the first textView ("Did you know?") and set textSize = "20sp".


5. Add a button that changes the fun fact displayed when clicked

In the Design tab, select a Button under Widgets, drag and position it on the phone display


In the Properties window on the right, set id = "nextButton", text = "Show Next Fun Fact", layout:width = "match_parent", layout:alignParent = [left, bottom], background = "#ffffff"


6. Change the background color of the activity

In the Component Tree window above Properties, select RelativeLayout, and change its background color to mauve, by setting background = "#c25975" to get the screen as below. Also set its id = "background"



7. Edit screen padding of our activity layout

Access the file dimens.xml to edit the layout dimensions as below
